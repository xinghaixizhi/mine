# 常用 VS Code 插件

### 代码相关

1. Prettier，美化代码
2. open in browser，在浏览器中打开
3. Live Server，本地开发热更新服务器
4. Chinese Lorem，简体中文乱数假文：
5. HTML CSS Support，HTML CSS 支持
6. CSS Peek，跳转到对应的选择器
7. Highlight Matching Tag，高亮匹配标签
8. TODO Highlight，高亮 TODO

#### Git

1. GitLens，增强 VS Code 自带的 Git 功能
2. Git Graph，可视化查看 Git Graph



### 界面美化

1. Chinese，汉化界面
